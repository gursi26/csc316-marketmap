## File Structure

### **index.html**: Our main HTML file, defining the structure and layout of our website.

### **/benefits vis**: A self-contained folder that contains all of the necessary code for the Benefits Visualization, including HTML, CSS, and JS files.

### **/css**: Contains all background image files as well as the global CSS stylesheet.

### **/dataset**: Contains all datasets and images used to fuel our visualizations.
- **/backgrounds**: Contains the background images for the Benifits Visualization
- **/building-icons**: Contains the building icons for the Map Visualization
- **/cleaned**: Contains the final cleaned datasets used for all visualizations
- **/guide-images**: Contains the images used in the info button for the visualizations
- **/Icons**: Contains the icons used for the benifits visualization
- **/logos**: Contains all company logos
- **/raw-data**: Contains the raw scraped data
- **/scripts**: Contains the scripts used for web-scraping and datset creation

### **/js**: Contains Javascript files for nearly all of the visualizations.
- **Map Visualization**: Contained in mapvis.js and main.js
- **Slope Chart Visualization**: Contained in slopeChart.js, bubbleRenderer.js, collisionDetection.js, constants.js, tooltipBarChart.js, and utils.js
- **Forest Visualization**: Contained in forest.js
- **Scatter Chart Visualization**: Contained in scatter.js
