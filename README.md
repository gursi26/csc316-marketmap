## File Structure

**index.html**: Our main HTML file, containing the code needed to jumpstart our website.

**/benefits vis**: A self-contained folder that contains all of the necessary code for the benefits visualization.

**/css**: Contains all background image files, as well as the global CSS stylesheet.

**/data** & **/dataset**: Contains all data used to fuel our visualizations.

**/fonts**: Contains fonts and related-images used as icons/symbols.

**/js**: Contains Javascript files for nearly all of the visualizations, as well as a static folder for relavant SVGs.
